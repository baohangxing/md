# Markdown

边写边看、开箱即用的在线 Markdown 编辑器。  
无需安装、无需注册 —— 打开网页就能用。

**立即体验：** https://baohangxing.github.io/md/

---

## 为什么好用

**即开即写**  
浏览器里直接编辑，所见即所得：左边写文档，右边马上看到成品效果。

**演示模式，专注阅读**  
一键进入全屏演示，适合汇报、分享、自己通读。右侧有目录，想跳到哪一章点一下就行；底部操作栏会自动收起，不挡视线。

**图表与公式也能看懂**  
流程图、时序图（Mermaid）、数学公式都可以正常展示，文档不再只有干巴巴的文字。

**表格好改**  
预览里的表格可以拖动调列宽，宽度会记住，下次打开还是你习惯的样子。

**界面随你**  
浅色 / 深色 / 跟随系统，白天晚上都舒适。编辑区和预览区可以单独藏起来，空间留给你真正在意的一侧。

**写着写着也不怕丢**  
内容自动保存在本机浏览器里，刷新、关掉标签页再回来，文档还在。

**随时带走**  
支持打开本地 `.md` 文件，也能一键下载 Markdown 或导出 HTML，方便备份和转发。

---

## 安全性

**文档留在你这台设备上**  
编辑内容只保存在本机浏览器里，不会上传到服务器，也不会用来登录或同步账号。

**没有账号、没有云端草稿箱**  
不收集注册信息，关页面也不必担心「云端还留着一份」。

**预览更安心**  
渲染前会过滤不安全内容，降低打开陌生 Markdown 时被恶意脚本影响的风险。

**你也可以完全本地使用**  
若更在意环境隔离，可以把页面当本地工具用：自己打开文件、自己下载带走，流程不经过第三方存储。

> 提醒：清空浏览器站点数据会一并清掉本地缓存；重要文档请记得下载备份。

## 适合谁用

- 写笔记、写周报、写 README 说明的同学  
- 需要边改边预览、偶尔做演示分享的人  
- 想轻量写文档、又不想装一整套软件的用户

## 开始使用

1. 打开 [在线地址](https://baohangxing.github.io/md/)  
2. 直接打字，或点「文件 → 打开」导入你的 Markdown  
3. 需要给别人看时，点「演示」进入全屏预览  

就这么简单。
